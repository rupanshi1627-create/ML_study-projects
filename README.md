#Linear Regression from Scratch

This project demonstrates how Linear Regression works internally without using any machine learning libraries.
-----------------
## What this project does
- Takes input data (hours studied)
- Calculates relationship between input and output
- Predicts marks using a mathematical model

------------------

##Concept Used

Linear Regression uses the formula:

y = mx + b

- m = slope (rate of change)
- b = intercept (starting point)
